DeoApp

A Web-app designed to build, customize, host, and monetize webcasting services. Creates customizable video-hosting websites with subscription base monetization options. Management tool allows for customization of all user owned websites, playlists, videos. Stat tracker shows scription count, video popularity, earnings and views. Subscription based video access with comments and social media integration. Back-end uses Python (Django) and MYSQL Database along with Paypal API for payment and Disqus API for social media integration and commenting, front-end uses HTML5 and CSS

Planetary N-Body Gravity Simulation

Multi-threaded planetary systems simulator that generates random gravitational systems. Implementation of a 4000+ planetary objects accelerated real-time gravity simulation using a hybrid Barnes-Hut model to tackle the n-body problem. Backend done with Java and front end uses JavaFX and Objectdraw API.

Multiplayer Snake

Real time web-app for instant user to user interaction in a endless game of multiplayer snake. Real time server to peer interaction, competitive game mechanics, leaderboard, etc. Back-end uses Javascript and express - node.js framework, Front-end designed with HTML and CSS3
